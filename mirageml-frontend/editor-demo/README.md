# MirageML Editor Demo

Визуальный конструктор сайтов с готовыми секциями (как на Tilda).

## 🚀 Быстрый старт

### Вариант 1: Через Node.js сервер

```bash
cd editor-demo
npm install
npm start
```

Откройте http://localhost:3002

### Вариант 2: Просто открыть файл

Дважды кликните на `index.html` в папке `editor-demo`

## 📦 Возможности

### ✅ Реализовано:

- **Библиотека секций** (13 готовых секций):
  - Hero (3 варианта)
  - Преимущества (2 варианта)
  - О компании
  - Команда
  - Цены
  - Контакты
  - Подвал

- **Drag-and-Drop**:
  - Перетаскивание из левой панели
  - Клик для быстрого добавления
  - Поиск по секциям
  - Фильтрация по категориям

- **Управление секциями**:
  - Перемещение вверх/вниз
  - Дублирование
  - Удаление
  - Выделение и редактирование

- **Панель слоёв**:
  - Отображение всех секций
  - Быстрое переключение между секциями
  - Управление порядком

- **Панель свойств**:
  - Редактирование отступов
  - Изменение цвета фона
  - Установка фонового изображения

- **Экспорт**:
  - Экспорт в HTML
  - Экспорт в CSS
  - Копирование в буфер
  - Предпросмотр результата

- **Горячие клавиши**:
  - `Ctrl+Z` — отменить
  - `Ctrl+Y` — повторить
  - `Delete` — удалить выделенную секцию

## 🎨 Добавление новых секций

Откройте `js/sections.js` и добавьте новую секцию:

```javascript
{
    id: 'my-section',
    name: 'Моя секция',
    category: 'hero',
    icon: 'fa-star',
    html: `
        <section class="section" style="padding: 80px 20px;">
            <!-- Ваш HTML -->
        </section>
    `,
    css: `
        .my-section {
            min-height: 500px;
        }
    `
}
```

Категории:
- `hero` — Hero секции
- `features` — Преимущества
- `about` — О компании
- `team` — Команда
- `pricing` — Цены
- `contact` — Контакты
- `footer` — Подвал

## 📝 Пример экспорта

### HTML:
```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>MirageML Project</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <section class="hero-section" style="...">
        <!-- Ваша секция -->
    </section>
</body>
</html>
```

### CSS:
```css
/* MirageML Project Styles */
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

.hero-section {
    min-height: 500px;
}
```

## 🐛 Известные ограничения

- Нет изменения размера секций (только отступы)
- Нет поворота элементов
- Нет экспорта в ZIP
- Сохранение только в localStorage
