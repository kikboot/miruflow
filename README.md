## ✨ Что такое MiruFlow?

**MiruFlow** — это инструмент для тех, кто хочет быстро и без лишних хлопот создавать веб-страницы. Представьте себе удобный графический редактор, где вы можете двигать блоки, менять тексты, настраивать цвета и сразу видеть готовый результат. 

Обычно конструкторы сайтов (вроде Tilda или Webflow) «привязывают» вас к себе: заставляют платить каждый месяц, иначе ваш сайт просто отключат. **В MiruFlow всё иначе.** Вы собираете сайт, нажимаете одну кнопку и скачиваете его чистый код. После этого сайт принадлежит только вам — его можно бесплатно запустить на любом хостинге навсегда.

---

## 🥰 Кому и чем поможет MiruFlow?

* **Фрилансерам:** Создавайте сайты для клиентов за считанные минуты. Экономьте время с помощью готовых шаблонов и не переплачивайте за сторонние платформы.
* **Веб-дизайнерам:** Быстро превращайте свои идеи и макеты в реальные рабочие веб-страницы прямо в браузере.
* **Начинающим разработчикам и студентам:** Используйте редактор как наглядное пособие. Вы можете менять элементы мышкой и сразу смотреть, из каких кирпичиков (HTML и CSS) состоит ваш сайт.
* **Малому бизнесу:** Возможность быстро собрать простой лендинг или визитку для проверки гипотез без бюджета на программистов.

## 💎 Главные фишки проекта

* **📦 Готовые шаблоны «из коробки»:** Не нужно начинать с пустого листа. Внутри уже есть стильные заготовки — выбирайте любую и редактируйте под себя прямо сейчас.
* **⚡️ Скачивание сайта в один клик:** Нажмите кнопку «Экспорт», и проект автоматически упакуется в аккуратный ZIP-архив со всеми картинками, шрифтами и стилями.
* **🧼 Никакого «мусора» в коде:** Редактор выдает чистый, легкий и аккуратный код. Такие сайты загружаются мгновенно и очень нравятся поисковикам (Яндекс и Google).
* **📱 Сайт всегда под рукой (В разработке):** Мы активно работаем над мобильной версией редактора. Совсем скоро вы сможете собирать и править свои сайты прямо с экрана смартфона или планшета в дороге.

## 🛠 На каком «топливе» работает проект?

Для тех, кому интересна техническая изнанка, MiruFlow построен на базе современных и надежных технологий:
* **Внешний вид:** Чистый JavaScript (Vanilla JS), HTML5, CSS3, шаблоны EJS
* **Серверная логика:** Node.js и Express.js (отвечают за сборку и скачивание ваших сайтов)
* **База данных:** PostgreSQL (надежное хранение шаблонов и элементов)

## 🚀 Как запустить проект у себя на компьютере?

Если вы хотите развернуть MiruFlow локально, выполните три простых шага:

1. **Скачайте код проекта:**
   ```bash
   git clone https://github.com
   cd miruflow
   ```

2. **Запустите сервер (Backend):**
   ```bash
   cd backend
   npm install
   node server.js
   ```

3. **Откройте интерфейс (Frontend):**
   Перейдите в папку `frontend` и просто откройте файл `index.html` в любом удобном браузере (или используйте плагин Live Server).
