Пользовательское соглашение

1. Общие положения

Настоящее Пользовательское соглашение (далее — «Соглашение») регулирует отношения между владельцем сервиса MiruFlow (далее — «Администрация») и пользователем (далее — «Пользователь»), возникающие при использовании сервиса MiruFlow.

2. Предмет соглашения

2.1. Администрация предоставляет Пользователю право использования визуального редактора MiruFlow для создания HTML/CSS страниц на условиях, определённых настоящим Соглашением.

2.2. Использование сервиса означает полное и безоговорочное принятие Пользователем всех условий настоящего Соглашения.

3. Права и обязанности сторон

3.1. Администрация обязуется:

  • Обеспеч��ть работоспособность сервиса MiruFlow
  • Обеспечить конфиденциальность персональных данных Пользователя
  • Оказывать техническую поддержку Пользователю

3.2. Пользователь обязуется:

  • Предоставлять достоверную информацию при регистрации
  • Не нарушать работоспособность сервиса
  • Не использовать сервис в противоправных целях
  • Не распространять контент, нарушающий законодательство РФ

4. Интеллектуальная собственность

4.1. Все права на сервис MiruFlow, включая программный код, дизайн, логотипы и товарные знаки, принадлежат Администрации.

4.2. Пользователь сохраняет все права на контент, созданный им с помощью сервиса.

4.3. Запрещается копирование, модификация и распространение программного кода MiruFlow без письменного разрешения Администрации.

5. Ответственность сторон

5.1. Администрация не несёт ответственности за:

  • Временные перебои в работе сервиса (технические работы, сбои)
  • Утрату данных Пользователя по техническим причинам
  • Действия третьих лиц (хакерские атаки, вирусы)

5.2. Пользователь несёт полную ответственность за контент, созданный с помощью сервиса.

6. Конфиденциальность и персональные данные

6.1. Администрация собирает и обрабатывает персональные данные Пользователя (email, имя) исключительно для предоставления услуг сервиса.

6.2. Администрация не передаёт персональные данные третьим лицам, за исключением случаев, предусмотренных законодательством РФ.

6.3. Пользователь вправе запросить удаление своих персональных данных, обратившись в поддержку.

7. Ограничения использования

7.1. Запрещается использовать сервис для создания и размещения:

  • Контента, нарушающего авторские права третьих лиц
  • Фишинговых страниц и вредоносных сайтов
  • Контента экстремистского, порнографического или незаконн��го характера

9. Изменение и расторжение соглашения

9.1. Администрация вправе в одностороннем порядке изменять условия настоящего Соглашения с уведомлением Пользователя.

9.2. Продолжение использования сервиса после изменений означает согласие Пользователя с новыми условиями.

9.3. Пользователь вправе отказаться от изменений, прекратив использование сервиса.

10. Срок действия

10.1. Настоящее Соглашение вступает в силу с момента его принятия Пользователем и действует бессрочно.

10.2. Соглашение может быть расторгнуто по инициативе Пользователя путём удаления аккаунта или Администрации в случае нарушения условий.
